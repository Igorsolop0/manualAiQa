Implement front-end logic to use linked `RegistrationSourceType[]` from `GET /GetClientByToken/`.

- Store in user state
- Use for login methods handling (unlink, display, validation)

## DevTools Monitoring Checklist

### Network Tab
Track these during Google auth flow:
- [ ] OAuth start request: URL, method, status
- [ ] Redirect to Google: 302 redirect chain
- [ ] Google callback: URL, parameters, status
- [ ] Final request after auth: URL, response
- [ ] Any 4xx/5xx errors
- [ ] Response body validation

**DO NOT LOG:**
- Access tokens
- Refresh tokens
- Client secrets
- Session IDs

### Console Tab
- [ ] JavaScript errors (red)
- [ ] Warnings (yellow)
- [ ] Failed network requests
- [ ] Token handling logs (should be redacted!)
